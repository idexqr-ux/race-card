InternalBuild notes (Ella)
  • Meeting / “Who is running this meeting”: Too generic. Prefer “Host Hunt” if that’s what you mean, or redirect to entered runners by race.
  • “Who is running in this race?”: If declarations not confirmed, answer with “The horses entered are …” (no question back).
  • Entries vs running: Use “runners/entries” for the printed card; “running/declared” only when day-of info is available.
  • Qualification hunt wording: If you prefer not to mention “Hunt for Bloodhounds”, store only the hunt name and omit the word “Hunt” in Excel if needed.
  • Placeholders: X values (e.g., weight, distance) are fine during build; keep spoken formats consistent.
  • “What”: The companion stays inside the same act of knowing, unless clearly invited out of it.
